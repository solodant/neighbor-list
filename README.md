# neighbor-list

**neighbor-list** — библиотека для поиска соседей в атомных системах на языке C.  
Реализует алгоритмы, аналогичные модулю `ase.neighborlist` из ASE (Atomic Simulation Environment).

---

## Требования

- Компилятор C (GCC, Clang, MSVC)
- Математическая библиотека (`-lm`)

---

## Установка

```bash
git clone https://github.com/solodant/neighbor-list.git # Клонирование репозитория
```
```bash
cd neighbor-list
```
